OpenAI

OpenAI es una empresa de investigación en inteligencia artificial con el objetivo de asegurar que el desarrollo de la IA beneficie a toda la humanidad.

Wikipedia

Wikipedia es una enciclopedia en línea, multilingüe y de acceso gratuito basada en un modelo wiki. Es una de las webs más populares en Internet.

GitHub

GitHub es una plataforma de desarrollo colaborativo de software que utiliza el sistema de control de versiones Git. Ofrece alojamiento de código para proyectos de desarrollo de software.

Stack Overflow

Stack Overflow es una comunidad en línea para programadores, donde pueden hacer preguntas, responderlas y aprender de otras personas.
