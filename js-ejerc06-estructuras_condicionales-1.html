Ejercicio 5 - Cadenas de Caracteres

Este script pide al usuario si quiere jugar. Si el usuario responde 's', se le pide el nombre de una fruta. Luego, segun la fruta, se muestra el precio por kilogramo. Finalmente, se le pregunta al usuario si hay algo más que quiera consultar. Si responde 'si', se muestra un mensaje mostrando que no hay tiempo para responder en ese momento, si no, salta un mensaje de agradecimiento. Si el usuario responde 'n' o cualquier otra cosa que no sea 's', se muestra un mensaje mostrando que es una pena que no quiera participar.
